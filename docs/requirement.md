# 需求
创建一个chrome插件, 提供多种数据的抓取功能，包括微博数据、YouTube视频。
# UI设计
1. 每个网站的抓取, 在popup上都有一个单独的tab
2. 不要用蓝紫色的配色
3. 页面元素要简洁，减少不必要的文案和配置项
4. 有进度条等元素来给用户及时的反馈

## 功能1 微博抓取功能
- 用户打开微博的列表，插件会自动滚动到页面的底部, 进行新的数据加载。然后从页面中的所有微博上一条一条开始抓取。每个微博包含作者、发布时间以及文本内容
- 输出的格式包含3种,列表，json，markdown
- 微博的示例URL: https://weibo.com/u/1769920597
- 在页面中有一个位置显示已经抓取到的json和markdown数据
- 增加一个排序和筛选功能，可以按关键词进行筛选。可以按发布的时间进行正序和倒序排列。
- 增加一个抓取到N条数据就结束的功能，避免无限制的页面滚动价值

## 功能2: 页面核心内容的markdown抓取
利用一些JS的库来识别页面中的核心内容，然后将他们抓取下来并转成Markdown的格式，要排除页面中的广告侧边栏等不重要的元素。


## 技术栈
- 前端技术栈:react+ts+pnpm+vite+tailwindcss+shadcnui
